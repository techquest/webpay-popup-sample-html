How to make the redirect page work!

Please note that the redirect pages specified in each of the popups will not work.

This is because WebPay does a POST to those pages and the ones provided here are merely html pages without servser side processing.

To make it work copy the files in this sample into your web application framework (i.e. PHP, ASP, ASP.Net, JSP etc) and handle the POST in the server code.
